🌿 Raízes do Bem
Plataforma web desenvolvida como parte da disciplina **Desenvolvimento Front-End para Web**, no curso de Análise e Desenvolvimento de Sistemas.
O projeto tem como objetivo aplicar os fundamentos de **HTML5, CSS3 e JavaScript**, criando um site acessível, responsivo e semântico voltado para ONGs.

---

👤 Autor
**Rafael Gonçalves Solera**

## 🚀 Tecnologias Utilizadas
- HTML5 semântico
- CSS3 responsivo (Flexbox + Grid)
- JavaScript (DOM, SPA, Validação)
- Git e GitHub (GitFlow, versionamento semântico)

---

## 🎯 Funcionalidades
- Cadastro de voluntários com validação de dados
- Navegação dinâmica (SPA)
- Modo escuro 
- Feedback visual interativo (toasts)
- Responsividade mobile e acessibilidade WCAG 2.1 AA
